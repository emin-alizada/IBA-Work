İmkanlar

Bizim filial və şöbələrimizdə özünəxidmət (self-service) məkanlarında, növbədə dayanmadan, əməliyyatçılara müraciət etmədən ödəniş terminallarında aşağıdakı əməliyyatları apara bilərsiniz:

  • Nağd vəsaiti kart hesabına yatırmaq
  • Kart hesabınızın balansını yoxlamaq və çıxarışı əldə etmək
  • Xidmət haqlarını ödəmək:
    • Mobil rabitə: müxtəlif mobil rabitə operatorlarının xidmət haqlarının ödənişi
    • İnternet: müxtəlif internet operatorlarının xidmət haqlarının ödənişi
    • TV: televiziya üzrə xidmət haqlarının ödənişi
    • Telefon: stasionar telefon rabitəsi üzrə xidmət haqlarının ödənişi
    • Kommunal xidmətlər: qaz, su, elektrik enerji haqlarının ödənişi
    • Digər: müxtəlif şirkətlərdə hesabınızın artırılması
    • Əyləncə: mərc oyunları saytlarında hesabınızın artırılması
  • InternetBank üçün birdəfəlik və 3DSecure parolları əldə etmək
  • SMS-xəbərdarlıq xidmətinə qoşulmaq 
  • MobilBank xidmətinə qoşulmaq 
  • KartTransfer xidməti vasitəsilə kartdan karta köçürmə aparmaq
  • Bank əməliyyatlarını aparmaq:
    • Hesaba pul köçürmək: cari hesaba pul köçürməsi və ya kredit ödənişi
